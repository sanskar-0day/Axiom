# Axiom OS Web Interface (`app/web`)

This directory contains the Svelte-based web interface and marketing site for the Axiom OS monorepo.

Axiom OS is a next-generation, Nix-based declarative operating system built with Nim and an embedded graph database.

The project is under active development.

## Deployment Root

Use `app/web` as the project root directory when deploying locally on your laptop or to Cloudflare Pages.

## Stack

- SvelteKit
- `@sveltejs/adapter-auto`
- GSAP (for cinematic landing-page motion)
